This is Pebble

Pebble serves as a starting point for static landing pages. Pebble is a customized version of the Yeoman Generator gulp-webapp. I created this repo as a starting point for static landing pages. Utilizing the Gulp Streaming Build system we can create well-crafted, modern landing pages quickly.

System Requirements
  1. NodeJS
  2. Bower
  3. Gulp
Getting Started

Once NodeJS is installed on your system, clone this repository in the directory you see fit.

Install Gulp-cli & Bower Globally

npm install gulp bower -g

Navigate to your directory in your terminal and install dependancies.

cd yourdirectory/ then npm install && bower install

Development Mode
  • gulp serve to start watching and serve app on localhost.
  • gulp buildto compile and build into /dist/.

Analytics Tracking Improved!

With the most recent update to Pebble, I've changed how to track Google Analytics Events using just HTML markup. In the new CoffeeScript files, you can see that the JS for Pebble has moved into an Event Delegated application system.

Simply add the class .send-ga-event to any element or form. The document listens for a click or submit event on any element with that class. Once fired, it sends an event directly to Google Analytics.

To customize this further, you will have to read through Google's Analytics.js Documentation.
